USER GUIDE

Este Dashboard contiene indicadores como:

  • TMO
  • AHT
  • Número de visitas

A su vez permite que los datos sean filtrados por Centro de Atención, rango de fechas y razón de visita.

El feeding de este reporte no se encuentra automatizado, por lo cual se indica a continuación como realizar este proceso.

Paso a Paso

  1. Solicitar Raw Data
  2. Pegar Data en Spread Sheet Base Raw Data File
  3. Arrastrar la fórmula de campos a calcular para que se actualicen los valores.
  4. Ir directamente al reporte CO KPI
